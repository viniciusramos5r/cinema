# cinema
simulador de compra de ingresso para cinema 

# Funcionalidades

- página de filmes com grid
- Escolha de tipo de ingresso (Inteira / Meia)
- calculo automático de valores
- sistema de login simples usando `sessionStorage`
- controle de acesso para compra (exige login)
- interface responsiva para desktop, tablet e celular (incompleta)
- estilo visual inspirado em sites de cinema
- seleção de assentos em sala de cinema
-
